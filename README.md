# DONCEZART

Creative portfolio website at [doncez.art](https://doncez.art) — design work, resources, and free assets.

## Tech Stack

SvelteKit 2 · Svelte 5 · Drizzle ORM · PostgreSQL · Vite 8

## Getting Started

```bash
pnpm install
cp .env.example .env   # fill in real values
pnpm dev               # starts on port 6969
```

## Building

```bash
pnpm build
pnpm preview
```
